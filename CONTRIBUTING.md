# Contributing to QNS Protocol

Thank you for your interest in contributing to the QNS Protocol Specification!

## How to Contribute

### Specification Feedback
- Open an issue for protocol clarifications or suggestions
- Submit pull requests for typo fixes or documentation improvements

### Implementation Questions
- For implementation guidance, see the specification document
- For SDK/API support, contact: dev@cr8os.cloud

### Security Issues
- Report security vulnerabilities privately to: security@cr8os.cloud
- Do NOT open public issues for security concerns

## Code of Conduct

Be respectful and constructive. We're building the future of the internet together.
